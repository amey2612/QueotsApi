# QuotesApi

A minimal .NET 8 Web API backed by EF Core + SQLite.

## Stack
- **ASP.NET Core 8** Minimal APIs
- **EF Core 8** with SQLite
- `IQuoteRepository` scoped DI
- Migrations applied at startup
- `ValidationProblemDetails` on bad input
- `ProblemDetails` on errors via exception middleware
- `CancellationToken` flows into all EF queries
- Structured logging via `ILogger<T>`

## Running Locally

```bash
# 1. Restore dependencies
dotnet restore

# 2. Create EF Core migration (first time only)
dotnet ef migrations add InitialCreate

# 3. Run — migrations auto-applied at startup
dotnet run
# API listens on https://localhost:5001 / http://localhost:5000
```

## Curl Test Commands (for submission)

Replace `http://localhost:5000` with your actual hos

```bash
# 1. POST — create a quote
curl -i -X POST http://localhost:5000/api/quotes \
  -H "Content-Type: application/json" \
  -d '{"author":"Marcus Aurelius","text":"You have power over your mind, not outside events."}'

# 2. GET list (page 1, size 10)
curl -i "http://localhost:5000/api/quotes?page=1&size=10"

# 3. GET by ID (use id returned from POST)
curl -i http://localhost:5000/api/quotes/1

# 4. DELETE
curl -i -X DELETE http://localhost:5000/api/quotes/1
```

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/api/quotes?page=N&size=N` | Paginated list |
| `POST` | `/api/quotes` | Create quote |
| `GET` | `/api/quotes/{id}` | Get by ID |
| `DELETE` | `/api/quotes/{id}` | Delete by ID |

Swagger UI available at `/swagger` in Development mode.
